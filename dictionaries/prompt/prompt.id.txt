# Indonesian Translation Continuation Task

You are continuing translation work on `dictionary/en_id.txt` — an English—Indonesian dictionary file.

Indonesian does **not** have grammatical gender for nouns. This dictionary uses simplified POS tags without gender markers.

---

## Your Task

1. **Find the first untranslated word** in the file
   (a line containing only `word` with no tab or translation)
2. **Translate in batches of 100 words** at a time
3. **Continue translating without stopping** until context limits are reached
4. **Format strictly** using the dictionary format defined below

---

## Dictionary Format

**File type:** TSV (Tab-Separated Values)
**Encoding:** UTF-8

**Structure:**

```
word<TAB>pos.translation@pos.translation
```

---

## Normalization Rules

* All headwords lowercase
* Hyphens removed (`any-thing → anything`)
* No commas used to separate meanings
* All distinct realizations separated with `@`
* No articles embedded in noun translations
* First occurrence of a headword wins

---

## Core Principle

> Indonesian does not have grammatical gender.
> Translations use simple POS tags without gender markers.

No sentence-level rewrite is performed.

## Meaning & POS Separation Rules (Authoritative)

* `@` separates **all distinct realizations**

  * different POS
  * different meanings (even within same POS)
* `,` is **not used**

### Examples

Single realization:

```
water n.air
```

Multiple meanings, same POS:

```
business  n.bisnis@n.perusahaan
```

Different POS:

```
control n.kontrol@v.mengontrol
```

Multiple meanings, same POS:

```
bank  n.bank@n.tepi
```

---

## Indonesian Language Notes

* **No grammatical gender** - nouns, pronouns, and adjectives do not inflect for gender
* **No articles** - Indonesian does not use articles like "the" or "a/an"
* **Verb affixes** - Use base form or appropriate affix (meN-, ber-, etc.) as natural
* **Plural forms** - Typically indicated by context or reduplication, use singular form
