## Dictionary Format

**File type:** TSV (Tab-Separated Values)
**Encoding:** UTF-8

**Structure:**

```
word<TAB>pos.translation@pos.translation
```

---

## Normalization Rules

* All headwords lowercase
* Hyphens removed (`any-thing → anything`)
* No commas used to separate meanings
* All distinct realizations separated with `@`
* No articles embedded in noun translations
* Bengali script must be used (not romanized)
* First occurrence of a headword wins

---

## Core Principle

> **Gender is a property of nouns.**
> Articles, pronouns, possessives, and adjectives expose **gender-bound realizations** to allow local binding.

No sentence-level rewrite is performed.

## Meaning & POS Separation Rules (Authoritative)

* `@` separates **all distinct realizations**

  * different POS
  * different genders
  * different meanings (even within same POS)
* `,` is **not used**

### Examples

Single realization:

```
water	n+m.জল
```

Same POS, different gender:

```
business	n+m.ব্যবসা@n+f.কোম্পানি
```

Different POS:

```
control	n+m.নিয়ন্ত্রণ@v.নিয়ন্ত্রণ করা
```

Same POS, same gender, different meanings:

```
bank	n+m.ব্যাংক@n+m.তীর
```

---

## Gender Rules (Bengali)

* **All Bengali nouns MUST include gender** (`n+f.` or `n+m.`)
* Use plain `n.` only if gender is genuinely unknown or not applicable
* Gender is **never inferred** for verbs or prepositions
* Bengali has **two genders**: masculine and feminine (no neuter)

---

## Agreement Carriers (Required)

### Articles

Bengali does not have definite articles like English "the". Indefinite articles:

```
a	art+m.একটি@art+f.একটি
an	art+m.একটি@art+f.একটি
```

Note: "একটি" (ekti) is the same form for both genders.

### Pronouns

```
it	pron+m.এটি@pron+f.এটি
he	pron+m.সে
she	pron+f.সে
they	pron.তারা
```

Note: Bengali uses "সে" (she) for both he and she in informal contexts.

### Possessives

```
my	pron+m.আমার@pron+f.আমার
your	pron+m.তোমার@pron+f.তোমার
his	pron+m.তার@pron+f.তার
her	pron+f.তার
```

### Adjectives

```
new	a+m.নতুন@a+f.নতুন
good	a+m.ভালো@a+f.ভালো
big	a+m.বড়@a+f.বড়
small	a+m.ছোট@a+f.ছোট
```

Note: Bengali adjectives do not typically inflect for gender.
