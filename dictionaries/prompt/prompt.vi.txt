## Dictionary Format

**File type:** TSV (Tab-Separated Values)
**Encoding:** UTF-8

**Structure:**

```
word<TAB>pos.translation@pos.translation
```

---

## Normalization Rules

* All headwords lowercase
* Hyphens removed (`any-thing → anything`)
* No commas used to separate meanings
* All distinct realizations separated with `@`
* No articles embedded in noun translations
* Vietnamese tone marks must be preserved
* First occurrence of a headword wins

---

## Core Principle

> Vietnamese does not have grammatical gender.
> Translations use simple POS tags without gender markers.

No sentence-level rewrite is performed.

---

## Meaning & POS Separation Rules (Authoritative)

* `@` separates **all distinct realizations**
  * different POS
  * different meanings (even within same POS)
* `,` is **not used**

### Examples

Single realization:

```
water n.nước
```

Multiple meanings, same POS:

```
business  n.kinh doanh@n.công ty
```

Different POS:

```
control n.sự kiểm soát@v.kiểm soát
```

Multiple meanings, same POS:

```
bank  n.ngân hàng@n.bờ
```

---

## Vietnamese Language Notes

* **No grammatical gender** - nouns, pronouns, and adjectives do not inflect for gender
* **No articles** - Vietnamese does not use articles like "the" or "a/an"
* **Classifiers** - Do not include classifiers (cái, con, etc.) in noun translations
* **Tone marks** - Must be properly preserved (à, á, ả, ã, ạ, etc.)
* **Compound words** - Use standard spacing (e.g., "kinh doanh" not "kinhdoanh")
