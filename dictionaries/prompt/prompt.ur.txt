# Urdu Translation Continuation Task

You are continuing translation work on `dictionary/en_ur.txt` — an English—Urdu dictionary file.

This dictionary **explicitly encodes grammatical gender for nouns** and **gender-bound realizations** for agreement carriers (articles, pronouns, possessives, adjectives).

---

## Your Task

1. **Find the first untranslated word** in the file
   (a line containing only `word` with no tab or translation)
2. **Translate in batches of 100 words** at a time
3. **Continue translating without stopping** until context limits are reached
4. **Format strictly** using the dictionary format defined below

---

## Dictionary Format

**File type:** TSV (Tab-Separated Values)
**Encoding:** UTF-8

**Structure:**

```
word<TAB>pos.translation@pos.translation
```

---

## Normalization Rules

* All headwords lowercase
* Hyphens removed (`any-thing → anything`)
* No commas used to separate meanings
* All distinct realizations separated with `@`
* No articles embedded in noun translations
* First occurrence of a headword wins

---

## Core Principle

> **Gender is a property of nouns.**
> Articles, pronouns, possessives, and adjectives expose **gender-bound realizations** to allow local binding.

No sentence-level rewrite is performed.

## Meaning & POS Separation Rules (Authoritative)

* `@` separates **all distinct realizations**
  * different POS
  * different genders
  * different meanings (even within same POS)
* `,` is **not used**

### Examples

Single realization:

```
water	n+m.پانی
```

Same POS, different gender:

```
business	n+m.کاروبار@n+f.تجارت
```

Different POS:

```
control	n+m.کنٹرول@v.قابو کرنا
```

Same POS, same gender, different meanings:

```
bank	n+m.بینک@n+m.کنارہ
```

---

## Gender Rules (Urdu)

* **All Urdu nouns MUST include gender** (`n+f.` or `n+m.`)
* Use plain `n.` only if gender is genuinely unknown or not applicable
* Gender is **never inferred** for verbs or prepositions

---

## Agreement Carriers (Required)

### Articles

Urdu does not have definite articles like English "the", but uses demonstratives:

```
the	art+m.یہ@art+f.یہ
a	art+m.ایک@art+f.ایک
an	art+m.ایک@art+f.ایک
```

### Pronouns

```
it	pron+m.یہ@pron+f.یہ
they	pron.وہ
he	pron+m.وہ
she	pron+f.وہ
I	pron.میں
you	pron.آپ@pron.تم
```

### Possessives

```
my	pron.میرا@pron.میری
your	pron.تمہارا@pron.تمہاری
his	pron+m.اس کا
her	pron+f.اس کی
```

### Adjectives

```
new	a+m.نیا@a+f.نئی
first	a+m.پہلا@a+f.پہلی
good	a+m.اچھا@a+f.اچھی
```
